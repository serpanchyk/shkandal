Виправ попередню відповідь моделі так, щоб вона була валідним JSON за схемою.

Не переосмислюй завдання і не додавай нових фактів. Збережи той самий зміст, але виправ формат, типи, назви полів і відсутні обов'язкові поля.

Суворо виконай обмеження схеми:

- видали елементи списків з неприпустимими типами або полями, якщо їх не можна
  виправити без вигадування;
- якщо список перевищує максимум, залиш лише найцентральніші елементи в межах
  максимуму;
- не вигадуй значення enum: використовуй лише значення, дозволені схемою;
- для картки статті з `is_case_candidate = false` обов'язково поверни
  `main_event_title_uk = null`, `events = []`, `entities = []` і
  `case_signature_terms = []`.

Поверни лише JSON.
