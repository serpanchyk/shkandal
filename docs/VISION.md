# Vision

Shkandal exists to preserve context around Ukrainian public scandals,
corruption investigations, political cases, institutional decisions, and other
socially important stories.

Instead of presenting a disposable news feed, the system turns source articles
into Ukrainian-language reader-facing dossiers with stable case summaries,
global entities, strict real-world events, chronological timelines, and visible
source provenance.

The product should feel live and modern rather than archival. The homepage
starts as a sortable case feed and can later grow into real-time event streams,
infographics, and trend views.

The MVP favors automatic publishing over manual review. It should still be
auditable: events and entities shown to readers must connect back to the source
articles that caused them to be added.
